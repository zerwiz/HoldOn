# Assets Directory

This directory contains all visual assets for the Hold On app.

## Required Assets

### App Icon
- **File**: `icon.png`
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Usage**: Main app icon for all platforms

### Android Adaptive Icon
- **File**: `adaptive-icon.png`
- **Size**: 1024x1024px
- **Format**: PNG
- **Usage**: Android adaptive icon (foreground layer)

### Splash Screen
- **File**: `splash.png`
- **Size**: 1242x2436px (or similar aspect ratio)
- **Format**: PNG
- **Usage**: App launch screen

### Favicon (Web)
- **File**: `favicon.png`
- **Size**: 48x48px (or 16x16px)
- **Format**: PNG or ICO
- **Usage**: Web browser favicon

## Creating Assets

### Design Guidelines

- **Minimalist**: Keep design simple and clean
- **Color Scheme**: Use the app's color scheme (green button, light background)
- **Typography**: Simple, readable fonts
- **Icon**: Could be a simple circle or hand icon representing "hold"

### Tools

- Use design tools like Figma, Adobe XD, or Canva
- Export at required sizes
- Ensure high quality (no pixelation)

## Placeholder Assets

For development, you can use placeholder images. For production, create proper branded assets.
